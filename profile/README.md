# NeoVoxel 团队

## 🚀 关于我们

NeoVoxel 是一支专注于 Minecraft Java 版插件开发的创新团队。我们致力于为服务器提供高性能、功能丰富且用户体验优秀的插件解决方案。

## 💡 技术栈

- **核心语言**: Java
- **开发框架**: 
  - Bukkit/Spigot/Paper API
  - Maven/Gradle 构建工具
- **数据库**: 
  - MySQL
  - SQLite
  - MongoDB
- **其他技术**:
  - NMS (Net Minecraft Server)
  - ProtocolLib
  - WorldEdit API
